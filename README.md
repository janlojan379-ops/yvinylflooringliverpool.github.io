# Luxury Vinyl Flooring Liverpool

A fully SEO-optimized GitHub Pages website targeting the keyword **"carpet and flooring shops near me"** with a contextual anchor link to [luxury vinyl flooring Liverpool](https://havenflooring.co.uk/).

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Main landing page — 1000+ word SEO article with schema markup, flooring type cards, FAQ section, and full design |
| `robots.txt` | Instructs search engine crawlers to index all pages and points to the sitemap |
| `sitemap.xml` | XML sitemap listing all indexable URLs for Google Search Console submission |
| `README.md` | This documentation file |

## SEO Details

- **Primary Keyword:** carpet and flooring shops near me
- **Anchor Text:** luxury vinyl flooring Liverpool
- **Target Link:** https://havenflooring.co.uk/
- **Anchor Placement:** Second paragraph of the introduction (high-relevance zone)
- **Schema Markup:** LocalBusiness structured data
- **Meta Tags:** Title, description, keywords, canonical, Open Graph
- **Content:** 1000+ words with semantic headings (H1 → H2 → H3), images with alt text, FAQ section, and internal linking

## Deployment

1. Upload all four files to your `luxuryvinylflooringliverpool` GitHub repository root
2. Enable GitHub Pages (Settings → Pages → Source: main branch)
3. Submit `sitemap.xml` to Google Search Console
4. Submit `sitemap.xml` to Bing Webmaster Tools
